## 프로젝트 의의
AWS로 웹서비스를 처음으로 배포
Github Actions로 CD(Continuous Deployment)
Docker로 클라이언트와 서버를 하나의 컨테이너로 묶어 배포
첫 Next.js 클라이언트 개발

## 유스케이스
- 카페 추가
- 리뷰 추가 / 삭제
- 주소로 카페 검색
- 이름으로 카페 검색
- 지도에서 위치로 카페 검색

## 개선이 필요한 부분
- 테스트 코드 추가
- AWS 기반 로드벨런싱 추가
- 클라이언트 SSR로 전환
- 카페 데이터 추가
